To run project - 

In command prompt UNIX at the directory of all files:
Compile:		javac OS.java
Run: 			java OS

After this a menu will display indicating which algorithm to run. 
select the algorithm and enter a quantum value when prompted to. 
program will run until 'q' is entered.
